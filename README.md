# 🚀 DevKit - Developer Utilities Toolkit

DevKit is a modern, responsive web application that brings together essential developer utilities in one place. Instead of opening multiple websites for simple tasks, DevKit provides a clean and efficient workspace for everyday development needs.



## ✨ Features

### 🔐 Password Generator
Generate strong and secure passwords with customizable options.

- Adjustable password length
- Include numbers
- Include symbols
- One-click copy functionality

### 📄 JSON Formatter
Format and beautify JSON instantly.

- Paste raw JSON
- Format with proper indentation
- Invalid JSON validation
- Clean readable output

### 📊 Word & Character Counter
Analyze text in real time.

- Word count
- Character count
- Sentence count
- Live updates while typing

### 🔤 Text Case Converter
Transform text into different formats.

- UPPERCASE
- lowercase
- Title Case
- Copy converted text

---

## 🎯 Why I Built This

As a web development student, I frequently use tools such as password generators, JSON formatters, and text utilities while coding and testing projects.

I wanted a single place where these commonly used utilities could be accessed quickly without switching between multiple websites.

DevKit was built to solve that problem.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/rohithssj/DevKit.git
```

Navigate to the project folder:

```bash
cd devkit
```

Open:

```bash
index.html
```

Or run using VS Code Live Server.
